# 🐍 Exercícios de Python | DIO + Accenture

Repositório com exercícios desenvolvidos durante o curso **Python para Análise e Automação de Dados**, realizado na plataforma **DIO (Digital Innovation One)** em parceria com a **Accenture**.

---

## 📚 Sobre o projeto

Este repositório reúne exercícios práticos com foco no desenvolvimento de:

- Lógica de programação
- Estruturas de dados
- Programação Orientada a Objetos (POO)
- Automação de tarefas
- Fundamentos de Python para análise de dados

Os conteúdos estão organizados por módulos, acompanhando a evolução do aprendizado.

---

## 🚀 Tecnologias utilizadas

- Python 3
- Git
- GitHub

---
## 🧠 Conteúdos abordados

- ✔ Tipos de dados (`int`, `float`, `str`)
- ✔ Estruturas condicionais (`if`, `elif`, `else`)
- ✔ Estruturas de repetição (`for`, `while`)
- ✔ Listas e manipulação de dados
- ✔ Métodos de listas
- ✔ Programação Orientada a Objetos (POO)
- ✔ Classes, objetos e herança
- ✔ Reutilização de código
