# 🐍 Exercícios de Python | DIO + Accenture

Repositório com exercícios desenvolvidos durante o curso **Python para Análise e Automação de Dados**, realizado na plataforma DIO (Digital Innovation One), em parceria com a Accenture.

---

## 📚 Sobre o projeto

Este repositório reúne exercícios práticos com foco no desenvolvimento da **lógica de programação**, **estrutura de dados** e aplicação do **Python para análise de dados e automação de tarefas**.

Os conteúdos estão organizados por módulos, acompanhando a evolução no aprendizado da linguagem.

---

## 🚀 Tecnologias utilizadas

* Python
* Git e GitHub
