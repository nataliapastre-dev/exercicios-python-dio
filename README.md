# 🐍 Exercícios de Python - DIO + Accenture

Repositório com exercícios desenvolvidos durante o curso **Python para Análise e Automação de Dados**, realizado na plataforma DIO (Digital Innovation One), em parceria com a Accenture.

## 📚 Sobre o projeto

Este repositório tem como objetivo reunir práticas e exercícios realizados ao longo do curso, com foco no desenvolvimento da lógica de programação e na utilização do Python para análise de dados e automação de tarefas.

## 🚀 Tecnologias utilizadas

* Python
* Git e GitHub

## 📂 Estrutura do repositório

```
fundamento/
 └── convertendo_tipos.py
```

## 🧠 Conteúdos abordados

* Conversão de tipos (int, float, str)
* Tipos de dados em Python
* Operações matemáticas
* Boas práticas de programação

## 🎯 Objetivo

Praticar conceitos fundamentais da linguagem Python, desenvolvendo uma base sólida para projetos futuros voltados à análise de dados e automação.
