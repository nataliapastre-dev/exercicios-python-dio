# 📊 Projeto SQLite com Python (DB-API)

Projeto prático para estudo de banco de dados com Python e SQLite, com foco em CRUD, organização em camadas, transações e boas práticas de segurança.

---

## 🚀 Objetivos

- Praticar operações com banco de dados (CRUD)
- Entender transações no SQLite
- Identificar riscos de SQL Injection
- Aplicar organização em camadas (domínio, serviço e banco)
- Desenvolver lógica com desafios práticos

---

## 🧰 Tecnologias utilizadas

- Python 3
- SQLite3 (biblioteca nativa)
- Dataclasses

---

## 📁 Estrutura do projeto
├── exercicio_dbapi.py # CRUD básico com SQLite
├── injecao_sql.py # Exemplo de SQL Injection (inseguro)
├── transacao.py # Controle de transações

├── 01_desafio/
│ ├── bd.py # Conexão e criação do banco
│ └── dominio.py # Modelos (Cliente, PF, PJ)

├── 02_desafio/
│ ├── bd.py # Banco de dados
│ ├── servico.py # Regras de negócio
│ └── main.py # Interface via terminal

└── meu_banco.sqlite # Banco de dados gerado


---

### 🗄️ CRUD com SQLite

- Consultar dados
