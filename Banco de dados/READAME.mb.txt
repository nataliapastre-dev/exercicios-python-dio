# 📊 Projeto SQLite com Python (DB-API)

Projeto prático para estudo de banco de dados utilizando **Python + SQLite**, com foco em operações CRUD, organização em camadas, transações e boas práticas de segurança.

---

## 🚀 Objetivos do projeto

- Praticar operações de banco de dados (CRUD)
- Entender o funcionamento de transações no SQLite
- Identificar e evitar riscos de SQL Injection
- Aplicar arquitetura em camadas (domínio, serviço e banco de dados)
- Desenvolver lógica de programação com desafios práticos

---

## 🧰 Tecnologias utilizadas

- Python 3
- SQLite3 (biblioteca nativa)
- Dataclasses

---

## 📁 Estrutura do projeto

```bash
├── exercicio_dbapi.py      # CRUD básico com SQLite
├── injecao_sql.py          # Exemplo de SQL Injection (inseguro)
├── transacao.py            # Controle de transações

├── 01_desafio/
│   ├── bd.py              # Conexão e criação do banco
│   └── dominio.py         # Modelos (Cliente, PF, PJ)

├── 02_desafio/
│   ├── bd.py              # Banco de dados
│   ├── servico.py         # Regras de negócio
│   └── main.py            # Interface via terminal

└── meu_banco.sqlite        # Banco de dados gerado

## 🧩 Desafios

### 🟢 Desafio 01

Implementar:

- `criar_conexao()`
- `criar_bd()`

**Objetivo:**
- Criar banco SQLite
- Criar tabela de clientes

---

### 🔵 Desafio 02

Sistema completo via terminal.

**Funcionalidades:**
- Criar cliente
- Listar clientes

**Arquitetura do sistema:**
- `dominio.py` → modelos (Cliente, PF, PJ)
- `bd.py` → banco de dados
- `servico.py` → regras de negócio
- `main.py` → interface via terminal
