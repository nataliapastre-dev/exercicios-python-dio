# 🚀 API com Flask - Sistema de Usuários e Contas

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Flask](https://img.shields.io/badge/Flask-Framework-black)
![Status](https://img.shields.io/badge/Status-Concluído-green)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

Projeto de API REST desenvolvido com Flask, aplicando boas práticas de arquitetura, organização de código e integração com banco de dados.

---

## 📌 Objetivo

Desenvolver uma API estruturada para:

- Gerenciar usuários e contas
- Aplicar arquitetura modular (controllers, models, config)
- Utilizar ORM com SQLAlchemy
- Trabalhar com migrações de banco de dados
- Implementar tratamento centralizado de erros
- Gerar documentação automática da API

---

## 🛠 Tecnologias

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Flask-Bcrypt
- Flask-Marshmallow
- APISpec (OpenAPI)
- SQLite (ambiente de desenvolvimento)
- PostgreSQL (produção via variável de ambiente)

---

## ⚠️ Tratamento de erros

### Casos tratados:
- Erros de integridade no banco
- Erros HTTP (400, 404, 409, etc.)

---

## 🔐 Segurança

- Criptografia de senha com Flask-Bcrypt
- Validação de dados com Marshmallow
- Controle centralizado de exceções

---

## 📊 Funcionalidades

- ✔ Cadastro de usuários
- ✔ Listagem de usuários
